# YourEnergy

Командний проєкт GoIT — веб-додаток для пошуку вправ за групами м'язів, частинами тіла та обладнанням.

🔗 **Живий сайт:** https://renk0rr.github.io/project-yourenergy/
🎨 **Макет Figma:** https://www.figma.com/design/1ifqGcQBIzMoc21yIqyV5q/YourEnergy
📋 **ТЗ:** https://docs.google.com/spreadsheets/d/18DqWibAs4rDYax5NldEsHO54dUgCtqgyy3So6LpLbyM

## Стек

Vite + Vanilla JS (ES modules) + Axios + iziToast

## Запуск

```bash
npm install
npm run dev
```

Відкриється http://localhost:5173.

## Скрипти

| Команда | Що робить |
|---|---|
| `npm run dev` | локальний dev-сервер |
| `npm run build` | production-збірка в `dist/` |
| `npm run preview` | preview зібраного білду |

## API

- Документація: https://your-energy.b.goit.study/api-docs
- **Всі запити — через `src/js/api.js`.** Не писати fetch/axios напряму.
